# smartphone-academic-analysis
A data science project exploring how smartphone habits impact teenage academic performance, comparing Linear Regression and Random Forest models.

About This Project

This project looks at how different smartphone usage patterns impact the academic performance of teenagers aged 14–18. Instead of just assuming "phones are bad," I analyzed survey data from 100 participants to track how specific habits—like study time, screen time, and active distraction levels—interact with final grades.

Key Highlights:

1) Data Exploration: Investigated direct effects like concentration breaks, alongside indirect trends like "Time Displacement" (where screen time actively replaces study hours).

2) Machine Learning Pipeline: Built and compared a baseline Linear Regression model against a Random Forest Regressor to predict student marks based on their behavioral habits.

3) Non-Linear Insights: The Random Forest model significantly outperformed Linear Regression (R^2 = 0.287 vs 0.130), proving that human habits are too complex and interconnected to be mapped by a simple straight-line equation.

4) Predictive Accuracy: The final model operates with a tight Mean Absolute Error (MAE) of 6.73 marks, making it a highly reliable baseline tool for evaluating student behavioral patterns.
